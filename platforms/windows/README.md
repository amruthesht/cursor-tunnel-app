# Windows

Download `CursorTunnelApp.exe` from [Releases](https://github.com/amruthesht/cursor-tunnel-app/releases) and run it. Config: `%APPDATA%\cursor-tunnel-app\`.

**From a clone:** `CursorTunnelApp.cmd` (needs Python 3.10+). `Launch-CursorTunnelApp.ps1` is called by the cmd file. `build-exe.ps1` produces `dist\CursorTunnelApp.exe`.
